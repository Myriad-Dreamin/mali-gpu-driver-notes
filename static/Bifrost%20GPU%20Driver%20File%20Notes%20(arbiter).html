Bifrost GPU Driver File Notes (arbiter)

20th August 2020 at 2:41pm

  • gpu/arm/midgard/arbiter/

    • mali_kbase_arbiter_interface.h: 定义arbiter_if_dev及其方法。

    • mali_kbase_arbiter_defs.h: 包含kbase_arbiter_vm_state和kbase_arbiter_device，kbase_arbiter_device是arbiter_if_dev和device的结合，而arbiter_if_dev在mali_kbase_arbiter_interface.h中且有所有相关的操作方法（函数指针）。

    • mali_kbase_arbif.h: 定义kbase_arbif_evt事件枚举类，以及kbdev为对象指针的各个对应操作函数。Mali arbiter interface APIs to share GPU between Virtual Machines。

    • mali_kbase_arbif.h: 定义kbase_vm_state虚拟机状态枚举类，看起来就很像qemu相关？以及kbdev为对象指针的各个对应操作函数。Mali arbiter power manager state machine and APIs
